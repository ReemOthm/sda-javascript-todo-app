# Todo App :notebook::iphone:

This project is a to do list application by applying HTML, CSS and Javascript.  The application allows users to manage their tasks effectively. 

## Todo App Functionalities

The application includes the following features:

1. Adding a Todo:
   - Display an input field where users can enter a new todo item with a description.
   - Included an "Add" button to add the entered todo item to the list.

2. Rendering Todo List:
   - Rendered the list of todos on the screen.
   - Each todo item display a checkbox and the task description.

3. Deleting a Todo:
   - Implemented a "Delete" button next to each todo item.
   - When the button is clicked, the corresponding todo item will remove from the list.

4. Editing a Todo:
   - Implemented an "Edit" button next to each todo item.
   - When the button is clicked, users will be allowed to update the task description of the corresponding todo.

5. Todo Counter:
   - Included a counter that displays the total number of todo items.

6. Persistent Storage:
   - Added the ability to persist todo items using local storage.
   - Ensured that the list remains intact even after the page is refreshed.

7. Search Functionality:
   - Implemented a search functionality that allows users to search for specific todo items based on keywords.

8. Clear Todo:
   - Implemented a "Clear" button to clear all the todos.
